Winter PennApps 2015 Open your door from your phone. This service allows a user to enter a passcode and use their photo to securely open his or her door.

Technology used:

  • Intel Edison
  • Debian (Ubilinux for Edison)
  • Node.js with Express
  • HTML/CSS/JavaScript/jQuery
  • Face recognition

Hardware list:

  • Intel Edison
  • Nema17 stepper motor
  • EasyDriver stepper motor driver
  • 12V power souce for stepper motor driver
  • Spool and box (we 3D printed these)

How to set up your Intel Edison's software

  1. Load Debian onto the Edison - tutorial
  2. Install libmraa on Ubilinux for Edison - tutorial
  3. Clone this repo.
  4. Type node app.js to start the server.
